# In-Row Edit Todo List - React.js Project

An advanced interactive Todo List application developed using React.js during my front-end development internship. This project focuses on component state management and seamless inline editing.

## 📅 Project Details
- **Role:** Front-End Development Intern
- **Timeline:** July 12, 2026 ( Internship Task)
- **Status:** Completed

## 🛠️ Technologies & Tools Used
- **React.js:** Functional components, conditional rendering, and hooks management.
- **JavaScript (ES6+):** Dynamic array manipulation (mapping, filtering, and updating state arrays).
- **CSS3:** Clean layout styling tailored and optimized for desktop/laptop displays.
- **Development Environment:** Visual Studio Code (VS Code)

## 📌 Features & Technical Highlights
- **In-Row Editing (`useState`):** Users can switch any specific row into an editable input state and save changes directly within the list.
- **Dynamic CRUD Operations:** Add, Read, Update, and Delete tasks in real-time using immutable React state updates.
- **Desktop First Layout:** Developed focusing strictly on laptop screen layouts as per internship project requirements.

*This project allowed me to deeply practice state arrays and component architecture in React, mastering how to handle direct inline user interactions efficiently.*
